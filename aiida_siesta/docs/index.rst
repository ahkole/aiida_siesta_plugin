Welcome to the AiiDA-Siesta documentation!
++++++++++++++++++++++++++++++++++++++++++++

..

The aiida-siesta python package interfaces the SIESTA DFT code (http://www.icmab.es/siesta) with the AiiDA framework (http://www.aiida.net).
The package contains: plugins for SIESTA itself and for other utility
programs, new data structures, and basic workflows.
It is distributed under the MIT license and available from
(https://github.com/albgar/aiida_siesta_plugin).
